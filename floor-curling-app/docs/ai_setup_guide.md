# AI 動作檢測：架設與環境規範指南 (SOP)

為了確保 AI 能夠準確判讀長輩的**手肘伸展度 (ROM)**、**軀幹穩定度**與**出手速度**，請務必遵守以下架設規則。

## 1. 拍攝位置與角度 (Camera Setup)

由於地板滾球 (Boccia) 適合亞健康長輩進行，AI 重點在於偵測「坐姿投擲」的穩定性，建議採用 **45度側前視角**。

### 📐 推薦角度：45度側前方
*   **右撇子長輩**：請將鏡頭架設在長輩的 **右前方 45度**。
*   **左撇子長輩**：請將鏡頭架設在長輩的 **左前方 45度**。

> **為什麼？** 
> *   **45度角** 可以完整看到投擲手臂的伸展 (不會被身體擋住)。
> *   同時能看到雙肩的水平狀態 (判斷軀幹是否歪斜)。
> *   純側面 (90度) 看不到另一邊肩膀；純正面 (0度) 手臂伸直時會重疊，無法計算角度。

### 📏 高度：與視線平行 (Eye Level)
*   請將手機/平板架設在約 **100cm - 120cm** 的高度。
*   **原則**：鏡頭高度應與長輩坐著時的**肩膀高度平行**。
*   ⛔ **禁止**：請勿由上往下俯拍 (高角度)，這會導致骨架變形，嚴重影響角度計算準確度。

### ↔️ 距離：全身入鏡 (2-3公尺)
*   距離長輩約 **2至3公尺**。
*   **畫面構圖**：確保畫面能容納長輩的 **頭頂到骨盆** (上半身完整入鏡)，且左右預留揮臂空間。

---

## 2. 環境要求 (Environment)

### 💡 光線 (Lighting)
*   **充足光源**：確保長輩臉部與身體正光。
*   **避免背光**：長輩背後不應有窗戶或強光，否則 AI 會偵測不到黑色剪影的骨架。

### 🏞️ 背景 (Background)
*   **單純背景**：背景越乾淨，AI 越準確。
*   **避免干擾**：避免背景有人員走動，或有與膚色/衣服顏色相近的物體。

---

## 3. 長輩坐姿規範 (Subject Positioning)

1.  **坐好坐滿**：請長輩臀部貼緊椅背，確保基礎坐姿穩定。
2.  **露出關節**：若穿著過於寬鬆的衣物 (如斗篷)，可能會遮擋手肘位置。建議穿著合身衣物，或在檢測時稍微捲起袖子露出手腕與手肘。

---

## 4. 操作流程 (Workflow)

### 🤖 混合分析模式（手動標記 + Session 平均值）

系統採用**混合方案**：以整段 Session 平均值為核心數據，搭配店長手動標記投球。

1. **選擇長者** — 在系統中選好要分析的長輩
2. **開啟攝影機** — 對準長輩，等待骨架出現

之後系統會自行完成以下工作：

| 狀態 | 系統行為 |
|------|---------|
| 👤 畫面中無人 | 顯示「等待長輩就位...」，**不記錄數據** |
| 🔵 長輩就位 | 自動開始分析骨架與動作（需偵測到雙肩+臀部+手臂） |
| 📊 即時分析 | HUD 持續顯示 ROM、軀幹傾斜、出手速度、進階指標 |
| 📌 手動標記 | 觀察到有效投擲時，按下「📌 標記這一球」記錄該瞬間指標快照 |
| 🛑 結束分析 | 按下結束後，系統計算整段 Session 平均值 + 顯示報告 |

### 📌 手動標記要點

- 每次按下「標記這一球」，系統會記錄**當下的 ROM、軀幹傾斜、出手速度、角速度**
- 有 **1 秒冷卻期** 防止誤觸
- 結束後報告會顯示每球的獨立數據比較

---

## 5. 簡易檢核表 (Checklist)

在按下「啟動 AI 檢測」前，請藥師/工作人員確認：

- [ ] **角度**：位於投擲手側前方 45 度嗎？
- [ ] **高度**：鏡頭是否與長輩肩膀等高？
- [ ] **光線**：是否沒有背光？
- [ ] **畫面**：長輩上半身是否完整在框內？
- [ ] **已選長者**：確認系統中已選擇正確的長輩？
- [ ] **無雜物**：鏡頭前方是否無其他人或物體干擾？

---

## 6. UI 輔助功能說明

系統畫面中的輔助線顏色代表意義：
*   🔵 **藍色骨架**：偵測良好，可進行測試。
*   🔴 **紅色警示**：偵測到危險動作 (如嚴重傾斜)。
*   ⚠️ **文字提示**：若出現「找不到人員」，請調整距離或光線。
*   📌 **標記反饋**：按下標記按鈕後顯示 toast 提示。

### 即時數據面板
*   三大核心指標（肘部 ROM、軀幹傾斜、出手速度）於畫面下方**每 0.5 秒更新一次**，方便觀察。
*   進階指標（中軸偏移、震顫、角速度、代償動作）**預設展開**顯示。
*   出手速度已精確至**小數點第二位**。

### 人體偵測守衛
*   系統需偵測到**雙肩 + 至少一側臀部 + 投擲手臂**才會開始分析。
*   避免鍵盤、桌子等非人體物體觸發誤偵測。
